# Frogodoro [WORK IN PROGRESS]
A frog-themed pomodoro timer. Planned to be used in conjunction with the "Pondering" Notes Progressive Web App (PWA).

For more detailed information about the project, head over to the Wiki pages.

## To Run
Locally:

1. Clone the repository 
```bash
git clone https://github.com/lnkl26/frogodoro.git
```

2. Run the `index.html`
    - You can double left-click the `index.html` in your file explorer, or
    - Use a live preview extension in your code editor, or
    - On Windows terminal, in the directory, run `start index.html`, or
    - On Linux terminal, in the directory, run `[browser] index.html`
        - ex. `firefox index.html` if you use Firefox as your browser.
